书签整理（五）正在使用的工具

dasein
2021-03-24 (Updated: 2025-08-30)
工具
收藏夹

之前写过一篇：那些在收藏夹里吃灰的工具
现在整理一下自己正在使用的工具吧：

Wox

一个快捷启动工具。推荐看sspai对此的介绍：Windows 上的 Alfred，免费开源的效率启动器：Wox
自带插件功能丰富：

  1. 快速启动应用，配合Everything可以快速检索文件
  2. 可快捷打开书签
  3. 查看天气（wpm install weather安装weather插件）
  4. 快捷搜索 如
    1. g + 搜索内容 会使用谷歌搜索
    2. sc + 搜索内容 使用谷歌学术搜索
    3. bd + 搜索内容 会使用百度搜索
    4. ….
  5. win+r快速打开cmd

Ditto

一个剪贴板管理软件。可以配置局域网共享和同步剪贴板。
win10自带剪贴板管理，Crtl+~触发

Quicker

一个功能极其强大的工具箱。

动作库中有大量用户自己制作的动作（和IOS的捷径性质相同，但功能更强大）
自带的功能如 EVER智识 集成了OCR 翻译等常用功能。可以用其搞一些自动化操作。

可以购买年付会员安装更多的插件。

ShareX

强大的截图和图片分享工具。

可以直接截图，然后将图片分享到各种云平台上，Onedrive Google Drive AWS S3等，以及各种图床如SM.MS等，并使用短链接转换工具分享图片。

Adgruad home

可以控制系统的DNS解析，支持DoH，DoT等DNS加密方式；可以通过DNS封锁清单：如常用的eastlist，里面包含大量屏蔽规则。
